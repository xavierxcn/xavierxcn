# Proposal: add-budget-target

## Why

用户在规划装修预算时，通常有一个总预算上限（如40万）。当前工具只能显示已填写项目的合计，无法直观看到距离目标预算还剩多少。添加目标预算功能可以帮助用户更好地控制支出。

## What Changes

### 1. 目标预算输入
- 在设置面板添加"目标预算"输入框
- 用户可输入总预算金额（如 400000）
- 目标预算持久化到 localStorage

### 2. 预算剩余显示
- 在汇总卡片中显示"剩余预算"
- 剩余预算 = 目标预算 - 已支出总计
- 当剩余为负数时（超支），用红色警示显示

### 3. 进度可视化
- 显示预算使用进度条
- 进度条颜色随使用比例变化（绿 → 黄 → 红）

## Impact

- **用户体验**：实时了解预算使用情况
- **决策支持**：帮助用户在预算范围内做出选择

## Scope

- 仅影响 `content/tools/decoration-budget/template.html`
- localStorage 版本保持 v4（新增 budgetTarget 字段）
