# Proposal: add-astrology-basics

## Why

当前占星版块只有星座、行星、宫位、相位四个分类，缺少基础知识介绍。新读者进入占星版块时，没有系统性的内容帮助理解占星术的全貌。

需要添加一个"基础入门"分类，介绍占星术的基本概念和背景知识。

## What Changes

新增 `basics`（基础入门）分类，包含以下文章：

### 1. 占星术简介 (intro.md)
- 占星术是什么
- 占星术的核心理念
- 占星术能做什么

### 2. 占星术的历史 (history.md)
- 古巴比伦的起源
- 希腊化时期的发展
- 阿拉伯黄金时代
- 现代占星术

### 3. 星盘基础 (chart-basics.md)
- 什么是星盘
- 星盘的构成：行星、星座、宫位、相位
- 星盘解读的基本逻辑

### 4. 占星术的分支 (branches.md)
- 本命占星
- 合盘占星
- 择日占星
- 流年推运
- 卜卦占星

### 5. 星座的分类 (sign-classifications.md)
- 阴阳：阳性星座与阴性星座
- 三分法：开创、固定、变动
- 四元素：火、土、风、水

## Scope

- 新增 `content/astrology/basics/` 目录
- 新增 `_category.yaml` 配置文件
- 新增 5 篇入门文章
- 基础入门显示在占星页面首位
